Install MacOS on VMware

为了在MacOS环境中做测试，准备在虚拟机VMware中安装一个苹果系统。

准备：

  • 解锁工具：macOS Unlocker。因为默认的VMware不支持安装macOS, 所以还需要解锁工具。
  • macOS系统镜像

镜像见百度盘：https://pan.baidu.com/s/1HYQjJvPN-0np2m402-Z2mg

提取码：apk4

解锁工具见：https://github.com/BDisp/unlocker/releases

解锁工具建议使用最新的，不用百度云盘里提供的。

安装：

见教程：https://www.cnblogs.com/xuexianqi/p/13463671.html

问题：

安装过程中遇到一个问题：

虚拟机提示：客户机操作系统已禁用CPU。请关闭或重启虚拟机。

找了不少解决方案，但是都无效。

大致有几个方向：

  1. 换电脑 :joy: 在AMD的CPU上玩虚拟机可能会出一些奇怪的无法解决的问题。
  2. 虚拟机设置的内存过低。
  3. VMware版本过低。
  4. 使用的镜像有问题，换镜像。
  5. 在启动时，选择：打开电源时进入固件，修改BIOS配置。
  6. 修改处理器的虚拟化引擎设置。
  7. 修改.vmx文件的配置

前面的方法多少有点不靠谱，试过一些仍无效之后，聚焦在最后一种方法上。但是看了几篇修改.vmx配置文件的文章，同质化太严重了，没有解决问题。

最后这篇文章马上解决问题：https://blog.csdn.net/weixin_42848413/article/details/115260253?spm=1001.2014.3001.5502

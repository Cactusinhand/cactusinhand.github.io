How to contribute to git upstream

作为目前最流行的开源分布式版本控制系统，Git的诞生超过17年，但是仍然具有非常活跃的开发者社区，其贡献者超过1.5K，这使得Git能够一直进行不断更新、优化各种功能。
但是要想参与到这个大型的国际化的开源社区中来并成为源码贡献者，对于国内大部分开发者来说并不是那么容易。
作为贡献者中的一员，我很乐意分享如何参与到这个开源社区的贡献者队伍来。

所以这个系列将从开发者的角度，分享我如何向Git官方社区提交代码。

截至目前，我向Git社区提交了4次代码：

  • 9096451acd (rev-list: support human-readable output for -disk-usage, 2022-08-11)
  • b0c4adcdd7 (remote-curl: send Accept-Language header to server, 2022-07-11)
  • 18337d406f (ls-files: update test style, 2022-07-03)
  • 4fe788b1b0 (builtin/clone.c: add –reject-shallow option, 2021-04-01)
